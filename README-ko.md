# Community Contracts v2.1.2

Community Contracts는 주민들과의 계약, 자동화, 온디맨드 서비스 등을 통해 Stardew Valley를 협력 기반의 번영하는 경제로 발전시키는 모드입니다.  
2.1.2 버전은 지금까지 가장 완성도 높고 플레이어 친화적인 업데이트로, 대화 중단 제거, 탐색 개선, 서비스 확장, 그리고 적합한 NPC를 더욱 쉽게 고용할 수 있도록 개선되었습니다.

## 시작하기
1. 아직 설치하지 않았다면 SMAPI를 설치하세요.  
2. Community Contracts를 다운로드하여 설치하세요.  
3. Stardew Valley를 실행하고 **V** 키를 눌러 Community Contracts 툴바를 엽니다.  
4. 툴바의 설정 메뉴에서 언제든지 단축키를 변경할 수 있습니다.  
5. 모든 기능, 서비스, 설정 옵션은 툴바에서 직접 접근할 수 있습니다.

## v2.1.2의 새로운 기능

### 대화 중단 제거
이제 모든 서비스는 툴바를 통해서만 이용할 수 있습니다. 즉:
- 서비스를 요청하기 위해 NPC를 클릭할 필요 없음  
- 게임 흐름을 끊는 팝업 대화창 없음  
- 주민과의 대화가 중단되지 않음  

전체적인 플레이 경험이 더욱 부드럽고 몰입감 있게 개선됩니다.

### 새로운 설정 메뉴 + 향상된 툴바 탐색
툴바가 더 명확하고 빠르게 사용하도록 재설계되었습니다:
- 전용 설정 메뉴 추가  
- 단축키, 서비스 옵션, 모드 설정을 즉시 변경 가능  
- 메뉴 간 이동이 더 직관적이고 깔끔해짐  

### 모든 NPC가 캐릭터 메뉴에 표시됨
우호도를 쌓을 수 있는 모든 주민(NPC 모드 포함)이 캐릭터 메뉴에 표시되며, 서비스 고용이 가능합니다.

메뉴에는 다음 정보가 표시됩니다:
- NPC 레벨  
- 우호도 포인트  
- 높은 우호도 순으로 자동 정렬  

### 동물 생산물 가공 서비스
NPC를 고용하여 다음을 가공할 수 있습니다:
- 달걀 → 마요네즈  
- 우유 → 치즈  
- 양털 → 천  
- 트러플 → 오일  

특징:
- 가공품 품질은 원재료와 동일  
- 가방 및 오토그랩버의 아이템 사용  
- 모든 축사 건물에서 작동  

### 주스 가공 서비스
NPC를 고용하여 다음을 주스로 가공할 수 있습니다:
- 과일  
- 채소  
- 일부 채집 아이템  

세부 사항:
- 주스 품질은 원재료와 동일  
- 생산량은 원재료 가치에 비례  
- 판매 시 항상 이익  
- 가방의 아이템 사용  

### 출하 상자로 직접 배송
모든 가공 서비스에서 다음 중 선택할 수 있습니다:
- 배송 상자(Delivery Chest)로 전달  
- 출하 상자(Shipping Bin)로 직접 배송하여 다음 날 수익 획득  

### 대규모 작업을 위한 다중 작업자 지원
일부 서비스는 여러 NPC가 동시에 작업할 수 있어 대규모 작업 속도가 크게 향상됩니다.

### 확장된 언어 지원 (총 11개 언어)
Community Contracts는 다음 언어를 지원합니다:
- 영어  
- 프랑스어  
- 이탈리아어  
- 독일어  
- 스페인어  
- 포르투갈어‑BR  
- 러시아어  
- 일본어  
- 한국어  
- 터키어 (신규)  
- 헝가리어 (신규)  

## v2.1.1의 기존 기능 (계속 포함됨)

### 농업 서비스
- 작물 심기  
- 경작  
- 물 주기  
- 화분 배치  
- 벌통 배치  
- 꿀 수확 + 선택적 미드(꿀술) 가공  
- 동물 돌보기  
- 씨앗 제작  

### 기타 서비스
- 사시미 제작  
- 광석 가공  
- 수액 채취기 설치  

### 추가 개선 사항
- 안정적인 작물 수확  
- 개선된 게잡이통 배치, 미끼 넣기, 수확  
- 직사각형 작업 범위 조절 가능  

### 핵심 시스템
- 색상 및 라벨을 사용자 지정할 수 있는 배송 상자  
- 우호도 및 기술에 따라 성장하는 계약 시스템  
- 전문 NPC 및 비전문 NPC 구분  
- 선택적 가공 시간 단축  

## 요약
Community Contracts v2.1.2는 대화 중단 제거, 탐색 개선, 자동화 확장, NPC 고용 최적화 등 가장 완성도 높은 업데이트입니다.  
작은 농장이든 대규모 농장이든, 이번 업데이트는 게임 플레이를 더욱 부드럽고 빠르고 몰입감 있게 만들어 줍니다.
