# Community Contracts v2.1.2

Community Contracts は、住民との契約・自動化・オンデマンドサービスを通じて、Stardew Valley を協力型の活気ある経済へと発展させる拡張です。  
バージョン 2.1.2 はこれまでで最も洗練され、プレイヤーに優しいアップデートです。会話の割り込みがなくなり、操作性が向上し、サービスが拡張され、最適な NPC をより簡単に雇えるようになりました。

## はじめに
1. まだの場合は SMAPI をインストールしてください。  
2. Community Contracts をダウンロードしてインストールします。  
3. Stardew Valley を起動し、**V** キーで Community Contracts ツールバーを開きます。  
4. ツールバーの設定メニューから、ホットキーはいつでも変更できます。  
5. すべての機能・サービス・設定オプションはツールバーから直接アクセスできます。

## v2.1.2 の新要素

### 会話の割り込みを撤廃
すべてのサービスがツールバー専用になり、次のような改善が行われました：
- サービス依頼のために NPC をクリックする必要がなくなる  
- ゲームプレイを中断するポップアップ会話が消える  
- 村人との会話が中断されなくなる  

よりスムーズで没入感のある体験になります。

### 新しい設定メニュー + ツールバー操作の改善
ツールバーがより分かりやすく、使いやすく再設計されました：
- 専用の設定メニューが追加  
- ホットキー・サービス設定・Mod 設定を即時変更可能  
- メニュー間の移動がより直感的に  

### すべての NPC がキャラクターメニューに表示されるように
友好度を持つすべての住民（Mod 追加 NPC を含む）がキャラクターメニューに表示され、サービスを依頼できます。

メニューには以下が表示されます：
- NPC レベル  
- 友好度ポイント  
- 友好度の高い順に自動ソート  

### 動物産品加工サービス
NPC を雇って以下を加工できます：
- 卵 → マヨネーズ  
- ミルク → チーズ  
- ウール → 布  
- トリュフ → オイル  

特徴：
- 加工品の品質は素材と同じ  
- 手持ちアイテムとオートグラバーから素材を使用  
- すべての畜産小屋に対応  

### ジュース加工サービス
NPC を雇って以下をジュースに加工できます：
- 果物  
- 野菜  
- 一部の採取物  

詳細：
- ジュースの品質は素材と同じ  
- 生成量は素材の価値に応じて増加  
- 販売時は常に利益が出る  
- 手持ちアイテムを使用  

### 出荷箱への直接配送
すべての加工サービスで以下を選択可能：
- 配送チェストに届ける  
- 出荷箱に直接配送して翌日に売上を得る  

### 大規模作業に複数の作業者を対応
一部のサービスでは複数 NPC の同時作業が可能になり、大規模な作業が大幅に高速化されます。

### 対応言語の拡張（全 11 言語）
Community Contracts は次の言語に対応しています：
- 英語  
- フランス語  
- イタリア語  
- ドイツ語  
- スペイン語  
- ポルトガル語（ブラジル）  
- ロシア語  
- 日本語  
- 韓国語  
- トルコ語（新規）  
- ハンガリー語（新規）  

## v2.1.1 から継続して含まれる機能

### 農業サービス
- 作物の植え付け  
- 土を耕す  
- 水やり  
- ガーデンポットの設置  
- 蜂の巣箱の設置  
- ハチミツの収集 + ミード加工（任意）  
- 動物の世話  
- 種の作成  

### その他のサービス
- 刺身の調理  
- 鉱石の加工  
- タッパー（樹液採取器）の設置  

### 追加改善
- 安定した収穫処理  
- カニかごの配置・餌入れ・収集の改善  
- 長方形の作業範囲を調整可能  

### コアシステム
- 色とラベルをカスタマイズできる配送チェスト  
- 友好度とスキルに応じて成長する契約システム  
- 専門職 NPC と非専門職 NPC  
- 加工時間の短縮オプション  

## まとめ
Community Contracts v2.1.2 は、会話の割り込み撤廃、操作性向上、自動化の拡張、NPC 雇用の最適化など、これまでで最も洗練されたアップデートです。  
小規模農場でも大規模経営でも、ゲーム体験がよりスムーズで快適で没入的になります。
