# Community Contracts v2.1.2

Community Contracts 通过与村民签订合同、自动化流程以及按需服务，将 Stardew Valley 打造成一个繁荣的合作型经济体系。  
2.1.2 版本是迄今为止最完善、最友好的更新——移除了对话打断、改进了导航、扩展了服务，并让雇佣合适的 NPC 比以往更加轻松。

## 入门指南
1. 如果尚未安装，请先安装 SMAPI。  
2. 下载并安装 Community Contracts。  
3. 启动 Stardew Valley，按下 **V** 键打开 Community Contracts 工具栏。  
4. 你可以随时在工具栏的设置菜单中更改快捷键。  
5. 所有功能、服务和配置选项都可直接通过工具栏访问。

## v2.1.2 的新增内容

### 移除对话打断
所有服务现在均通过工具栏访问，这意味着：
- 不再需要点击 NPC 来请求服务  
- 不再有打断游戏流程的对话弹窗  
- 不再出现与村民的对话被中断的情况  

整体体验更加流畅、快速且沉浸。

### 全新的设置菜单 + 改进的工具栏导航
工具栏经过重新设计，更加清晰高效：
- 新增专用设置菜单  
- 可即时调整快捷键、服务选项和模组偏好  
- 导航更直观、更易用  

### 所有 NPC 现在都会显示在角色菜单中
所有能够获得友好度的村民（包括模组添加的 NPC）现在都会显示在角色菜单中，并可被雇佣。

角色菜单显示：
- NPC 等级  
- 友好度点数  
- 自动按友好度从高到低排序  

### 动物产品加工服务
雇佣 NPC 来加工以下物品：
- 鸡蛋 → 蛋黄酱  
- 牛奶 → 奶酪  
- 羊毛 → 布料  
- 松露 → 松露油  

特点：
- 加工品品质与原材料一致  
- 使用背包和自动收集器中的物品  
- 适用于所有畜舍建筑  

### 果蔬榨汁服务
雇佣 NPC 将以下物品加工为果汁：
- 水果  
- 蔬菜  
- 部分采集物  

详情：
- 果汁品质与原材料一致  
- 产量基于物品价值  
- 出售时始终有利润  
- 使用背包中的物品  

### 直接配送至出货箱
所有加工服务均可选择：
- 配送至你的配送箱，或  
- 直接配送至出货箱，次日获得收益  

### 大型任务可由多名工人同时完成
部分服务现在支持多个 NPC 同时工作，大幅提升大型任务的效率。

### 扩展语言支持（共 11 种语言）
Community Contracts 现已支持：
- 英语  
- 法语  
- 意大利语  
- 德语  
- 西班牙语  
- 葡萄牙语‑BR  
- 俄语  
- 日语  
- 韩语  
- 土耳其语（新增）  
- 匈牙利语（新增）

## 总结
Community Contracts v2.1.2 是迄今为止最完善的版本——移除对话打断、改进导航、扩展自动化，并让 NPC 雇佣更加直观。  
无论你经营的是小型农场还是大型农业基地，这次更新都能让你的游戏体验更加流畅、高效、沉浸。
